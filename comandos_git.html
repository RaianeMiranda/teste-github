Configurações do git

comandos quando intalamos o git em um computador

  • git add nome_do_arquivo : insere os arquivos para serem versionados;
  • git commit -m "descreva o que foi feito" : cria uma versão de um ou mais arquivos (adicionados com o comando git add), com uma descrição do que foi feito;
  • git status : verifica se há alguma modificação na pasta do projeto;
  • git branch -m
  • git config --global user.email "coloque seu email": configura o email cadastrado no
  • git push origin main
  • git log
  • git push

Comandos para criar versões
  • git add .: adiciona todos arquivos alterados/modificados/excluídos para serem monitorados pelo github, em termos
